# Rouzax Repository

A personal Kodi addon repository for installing and auto-updating Rouzax addons.

## Installation

1. Download the latest `repository.rouzax-X.Y.Z.zip` from [Releases](https://github.com/Rouzax/repository.rouzax/releases)
2. In Kodi, go to **Add-ons** > **Install from zip file**
3. Select the downloaded zip
4. The repository is now installed - find addons under **Install from repository** > **Rouzax Repository**

## Included Addons

| Addon | Description |
|-------|-------------|
| [EasyMovie](https://github.com/Rouzax/script.easymovie) | Simplify movie night with a filter wizard and curated random selection |
| [EasyTV](https://github.com/Rouzax/script.easytv) | No scrolling. No deciding. Just watching. |
| [QuickList](https://github.com/Rouzax/script.quicklist) | Context menu addon for managing TV shows in smart playlists |
